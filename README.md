# O-Autofish-NTE-android
Авторыбалка на android для игры neverness to everness
Вы делаете это на свой страх и риск, хых.

Нужно установить AutoJs6 (https://github.com/SuperMonster003/AutoJs6/releases).
Дать ему все нужные разрешения. 

В скипте O-Autofish-NTE-android есть строки настроек. Скорее всего вам нужно их поменять под свои. Узнать координаты можно через AutoJs6, включить в левой шторке расположение указателя. Через macrorify в игре узнать можно цвет. Не на скриншоте, а именно в игре надо словить.
